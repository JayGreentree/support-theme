# Themedost

Themedost is a free, responsive and stylish osTicket "client-side" theme.  

Though I call it a theme, this is rather a styled modification of the osTicket software itself. 
You can install it just like you would install the normal osTicket software. 

Please note that only the client side is customised. The admin side is un touched.

Check out a [Live Preview](https://jinsonathemes.com/ost/themedost/).
